# EI Web Data Project

## Structure du Projet

```
ei_wd/
├── datasets/          # Jeux de données collectés
├── étiquettage/       # Scripts et outils d'annotation des données
├── ML/               # Modèles de machine learning et algorithmes
├── results/          # Résultats des analyses et modèles
├── scrapping/        # Scripts de collecte de données web
└── README.md


## Description

Ce projet combine la collecte de données web, l'étiquettage, et l'apprentissage automatique pour analyser des données extraites du web.

## Composants

- **Scrapping**: Extraction automatisée de données depuis le web
- **Datasets**: Stockage des données collectées
- **Étiquettage**: Annotation et préparation des données
- **ML**: Implémentation des modèles d'apprentissage automatique
- **Results**: Analyses et résultats obtenus
